Converts a RGB image to YUV.

This function converts a RGB image to YUV by extracting the RGB values of each pixel and calculating the YUV values. The YUV values are calculated using the formulas:
Y = 0.299 * R + 0.587 * G + 0.114 * B
U = -0.14713 * R - 0.28886 * G + 0.436 * B + 128
V = 0.615 * R - 0.51499 * G - 0.10001 * B + 128
where R, G, and B are the RGB values of the pixel.

The main function takes in command line arguments to determine the operations to be performed on the input image or video. The supported arguments are as follows:
–source <inputPath>: Specifies the path to the input image or video file.
–watermark <watermarkPath>: Specifies the path to the watermark image file. If provided, the watermark will be applied to the image or video.
–histogram: Calculates and displays the histogram of the image or video.
–blur <maxkernel>: Applies a blur effect to the image or video using both normal blur and Gaussian blur. The maxkernel parameter specifies the maximum kernel size for the blur effect.
–segmentation <thresholdvalue>: Applies segmentation to the image or video using manual thresholding. The thresholdvalue parameter specifies the threshold value for segmentation.
–grayscaled: Converts the image or video to grayscale.
–YuV: Applies the YUV color space conversion to the image or video.
–HistEq: Applies histogram equalization to the grayscale image or video.
The main function first parses the command line arguments and sets the corresponding flags and parameters. It then checks the file extension of the input file to determine if it is an image or a video. If it is an image, the image is read using OpenCV's imread function. If it is a video, the video is read using OpenCV's VideoCapture class and processed frame by frame.
For each frame, the specified operations are applied in the following order:
If the –watermark flag is set, the watermark is applied to the frame using the applyWatermark function.
If the –grayscaled flag is set, the frame is converted to grayscale using the grayScaleConversion function.
If the –YuV flag is set, the YUV color space conversion is applied to the frame using the applyYuV function.
If the –histogram flag is set, the histogram of the frame is calculated and displayed using the displayHistogram function.
If the –blur flag is set, a blur effect is applied to the frame using both normal blur and Gaussian blur with the specified maxkernel using the applyBlur function.
If the –segmentation flag is set, segmentation is applied to the frame using manual thresholding with the specified thresholdvalue using the applySegmentation function.
If the –HistEq flag is set, histogram equalization is applied to the grayscale frame using the applyHistogramEqualization function.
After processing all frames, the resulting image or video is displayed using OpenCV's imshow function. If the –watermark flag is set, the watermark is also applied to the resulting image using the applyWatermark function. Finally, the program waits for a key press and returns 0 if the program executed successfully.
