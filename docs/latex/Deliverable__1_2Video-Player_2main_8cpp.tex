Main function for video processing. 

This function takes a video file path as an argument, opens the video file, and performs various operations on the video frames. It converts the frames from BGR to YUV, performs 4\+:2\+:2 and 4\+:2\+:0 subsampling, and converts the frames back to RGB. The processed frames are displayed in separate windows.
