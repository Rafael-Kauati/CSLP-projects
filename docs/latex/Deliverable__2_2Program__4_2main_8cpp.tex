Main function for video processing. 

~\newline
 This function takes a video file path as an argument, opens the video file, and performs various operations on the video frames. ~\newline
 It allows the user to choose between RGB and YUV color modes and to set the speed of the video playback. ~\newline
 The processed frames are displayed in a window.
