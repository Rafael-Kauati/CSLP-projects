Function to solve the Deliverable 1, Program 3 Image Processment operations. 

This function loads an image and a watermark, applies the watermark to the image, calculates histograms for each channel, converts the image to YUV and grayscale, calculates the histogram of the grayscale image, applies histogram equalization, applies a Gaussian blur, and applies a threshold.

\begin{DoxyReturn}{Returns}
int Returns 0 on successful execution, -\/1 if the image or watermark could not be loaded.
